Ecmascript

Ecmascript (es) est une norme sur laquelle s'appuie JS. (Actionscript => flash)

La dernière version ES es7-8-9 (très peu de différence avec es6)

  • es1 - 1997
  • es2 - 1999
  • es4 - version abandonnée
  • es5 - 2009
  • es6 - 2015
  • es7 - 2017
  • ...

Jusqu'en 2010, JS était mal considéré par les développeurs ( peu normé, troppermissif )

L'arrivée de Google Chrome, avec en plus l'envie de Google de refaire fonctionner correctement JS. Création du moteur d'interprétation V8. => un boost des capacitées de JS et son éxécution

Grâce au V8, des nouveaux langages et nouvelles méthodes de développement sont arrivées, notemment NodeJs (2009), qui s'appuis sur JS mais s'éxécute sur le serveur

La convention du code en es6 est visible sur Github es6
